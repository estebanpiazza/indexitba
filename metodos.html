Métodos Comunes de Arrays

Aquí tienes el `
    ` actualizado con un resumen de los parámetros que recibe cada método: ```html
    • map(): Aplica una función a cada elemento del array y devuelve un nuevo array.
      Parámetros: callback(elemento, índice, array).
    • filter(): Crea un nuevo array con los elementos que cumplen una condición.
      Parámetros: callback(elemento, índice, array).
    • reduce(): Aplica una función a cada elemento y reduce el array a un único valor.
      Parámetros: callback(acumulador, elemento, índice, array), valorInicial (opcional).
    • forEach(): Ejecuta una función en cada elemento del array (sin devolver nada).
      Parámetros: callback(elemento, índice, array).
    • find(): Devuelve el primer elemento que cumple una condición.
      Parámetros: callback(elemento, índice, array).
    • findIndex(): Devuelve el índice del primer elemento que cumple una condición.
      Parámetros: callback(elemento, índice, array).
    • includes(): Verifica si un valor está presente en el array.
      Parámetros: valorBuscado, índiceDesde (opcional).
    • some(): Verifica si al menos un elemento cumple una condición.
      Parámetros: callback(elemento, índice, array).
    • every(): Verifica si todos los elementos cumplen una condición.
      Parámetros: callback(elemento, índice, array).
    • push(): Añade uno o más elementos al final del array.
      Parámetros: elemento1, elemento2, ..., elementoN.
    • pop(): Elimina y devuelve el último elemento del array.
      Parámetros: Ninguno.
    • shift(): Elimina y devuelve el primer elemento del array.
      Parámetros: Ninguno.
    • unshift(): Añade uno o más elementos al principio del array.
      Parámetros: elemento1, elemento2, ..., elementoN.
    • slice(): Devuelve una copia superficial de una porción del array.
      Parámetros: inicio, fin (opcional).
    • splice(): Añade, elimina o reemplaza elementos en el array.
      Parámetros: índiceInicio, númeroElementosAEliminar (opcional), elementosAInsertar.
    • concat(): Combina dos o más arrays y devuelve un nuevo array.
      Parámetros: array1, array2, ..., arrayN.
    • join(): Combina todos los elementos del array en un string.
      Parámetros: separador (opcional).
    • reverse(): Invierte el orden de los elementos del array.
      Parámetros: Ninguno.
    • sort(): Ordena los elementos del array según una función de comparación.
      Parámetros: funciónComparación (opcional).
    • flat(): Devuelve un nuevo array con todos los sub-arrays concatenados hasta la profundidad especificada.
      Parámetros: profundidad (opcional, por defecto es 1).
    • flatMap(): Aplica una función a cada elemento y aplana el resultado en un nuevo array.
      Parámetros: callback(elemento, índice, array).
    ```

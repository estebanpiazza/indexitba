📊 Excel Cheat Sheet

🕒 Funciones de Fecha y Hora

=AHORA() 👉 Fecha y hora actuales =HOY() 👉 Fecha actual =AÑO(FECHA) 👉 Año de una fecha =MES(FECHA) 👉 Mes de una fecha =DIA(FECHA) 👉 Día de una fecha =DIASEM(FECHA;1) 👉 Día de la semana (1=lunes) =HORA(NÚMERO DE SERIE) 👉 Hora de una fecha/hora =MINUTO(NÚMERO DE SERIE) 👉 Minuto de una fecha/hora =SEGUNDO(NÚMERO DE SERIE) 👉 Segundo de una fecha/hora =FECHA(AÑO; MES; DÍA) 👉 Crea una fecha

🔢 Funciones Matemáticas

=ALEATORIO.ENTRE(INFERIOR; SUPERIOR) 👉 Número aleatorio entre límites =ATAN(NÚMERO) 👉 Arcotangente =COS(NÚMERO) 👉 Coseno =SENO(NÚMERO) 👉 Seno =TAN(NÚMERO) 👉 Tangente =EXP(NÚMERO) 👉 e elevado a un número =LN(NÚMERO) 👉 Logaritmo natural =LOG(NÚMERO; BASE) 👉 Logaritmo en base específica =LOG10(NÚMERO) 👉 Logaritmo en base 10 =RAÍZ(NÚMERO) 👉 Raíz cuadrada =ENTERO(NÚMERO) 👉 Redondear hacia abajo =DECIMAL(NÚMERO; DECIMALES) 👉 Redondear a X decimales =PI() 👉 Valor de π =FACT(NÚMERO) 👉 Factorial

🧮 Funciones Estadísticas

=PROMEDIO(RANGO) 👉 Media =MEDIANA(RANGO) 👉 Mediana =MODA(RANGO) 👉 Moda =VAR(RANGO) 👉 Varianza (muestra) =VARP(RANGO) 👉 Varianza (población) =DESVEST(RANGO) 👉 Desviación estándar (muestra) =DESVESTP(RANGO) 👉 Desviación estándar (población) =FRECUENCIA(DATOS; GRUPOS) 👉 Distribución de frecuencia =COEF.DE.CORREL(MATRÍZ_1; MATRÍZ_2) 👉 Correlación

🔍 Funciones de Búsqueda y Referencia

=BUSCARV(VALOR; MATRÍZ; COLUMNA; TIPO) 👉 Búsqueda vertical =BUSCARH(VALOR; MATRÍZ; FILA; TIPO) 👉 Búsqueda horizontal =COINCIDIR(VALOR; MATRÍZ; TIPO) 👉 Posición de un valor =INDICE(MATRÍZ; FILA; COLUMNA) 👉 Valor en una posición

🧠 Funciones Lógicas

=SI(PRUEBA; SI_VERDADERO; SI_FALSO) 👉 Condicional =SI.ERROR(VALOR; VALOR_SI_ERROR) 👉 Manejar errores =Y(cond1; cond2;...) 👉 Todas las condiciones verdaderas =O(cond1; cond2;...) 👉 Al menos una condición verdadera =ESBLANCO(VALOR) 👉 ¿Es celda vacía? =ESERROR(VALOR) 👉 ¿Es error?

🅰️ Funciones de Texto

=CONCATENAR(TEXTO1; TEXTO2;...) 👉 Unir textos =ESPACIOS(TEXTO) 👉 Eliminar espacios extra =IZQUIERDA(TEXTO; N) 👉 N primeros caracteres =DERECHA(TEXTO; N) 👉 N últimos caracteres =EXTRAE(TEXTO; INICIO; N) 👉 Extraer parte de un texto =LARGO(TEXTO) 👉 Largo de un texto =MINUSC(TEXTO) 👉 A minúsculas =MAYUSC(TEXTO) 👉 A mayúsculas =NOMPROPIO(TEXTO) 👉 Capitalizar =SUSTITUIR(TEXTO; ORIGINAL; NUEVO; N) 👉 Reemplazo controlado =REEMPLAZAR(TEXTO; INICIO; N; NUEVO) 👉 Reemplazo por posición =TEXTO(VALOR; FORMATO) 👉 Dar formato personalizado =HALLAR(TEXTO; DENTRO; INICIO) 👉 Buscar posición (sensible a mayúsculas) =ENCONTRAR(TEXTO; DENTRO; INICIO) 👉 Buscar posición (NO sensible)

🧠 Funciones de Matrices

=MMULT(MATRÍZ1; MATRÍZ2) 👉 Multiplicación de matrices =MDETERM(MATRÍZ) 👉 Determinante =MINVERSA(MATRÍZ) 👉 Matriz inversa =TRANSPONER(MATRÍZ) 👉 Transponer filas ↔ columnas

📋 Funciones de Conteo y Sumatoria

=CONTAR(RANGO) 👉 Cuenta números =CONTAR.SI(RANGO; CRITERIO) 👉 Cuenta según condición =CONTARA(RANGO) 👉 Cuenta no vacíos =SUMA(RANGO) 👉 Suma total =SUMAR.SI(RANGO; CRITERIO; RANGO_SUMA) 👉 Suma condicionada =PROMEDIO.SI(RANGO; CRITERIO; RANGO) 👉 Promedio condicionado

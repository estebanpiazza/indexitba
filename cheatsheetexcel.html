📊 Excel Cheat Sheet

🕒 Funciones de Fecha y Hora

=AHORA() =NOW() 👉 Devuelve la fecha y hora actuales =HOY() =TODAY() 👉 Devuelve solo la fecha actual =AÑO(fecha) =YEAR(date) 👉 Extrae el año de una fecha =MES(fecha) =MONTH(date) 👉 Extrae el mes de una fecha =DIA(fecha) =DAY(date) 👉 Extrae el día de una fecha =DIASEM(fecha; tipo) =WEEKDAY(date, type) 👉 Devuelve el día de la semana (1=lunes, 2=martes, etc.) =HORA(valor_hora) =HOUR(time_value) 👉 Devuelve la hora de una hora =MINUTO(valor_hora) =MINUTE(time_value) 👉 Devuelve el minuto de una hora =SEGUNDO(valor_hora) =SECOND(time_value) 👉 Devuelve el segundo de una hora =FECHA(año; mes; día) =DATE(year, month, day) 👉 Crea una fecha a partir de año, mes y día

🔢 Funciones Matemáticas

=ALEATORIO.ENTRE(inferior; superior) =RANDBETWEEN(bottom, top) 👉 Número aleatorio entre los valores que se indican =ATAN(número) =ATAN(number) 👉 Devuelve la arcotangente de un número =COS(número) =COS(number) 👉 Devuelve el coseno del ángulo dado en radianes =SENO(número) =SIN(number) 👉 Devuelve el seno del ángulo dado en radianes =TAN(número) =TAN(number) 👉 Devuelve la tangente del ángulo dado en radianes =EXP(exponente) =EXP(power) 👉 Devuelve e elevado a la potencia dada =LN(número) =LN(number) 👉 Devuelve el logaritmo natural de un número =LOG(número;base) =LOG(number, base) 👉 Devuelve el logaritmo de un número en la base indicada =LOG10(número) =LOG10(number) 👉 Devuelve el logaritmo base 10 de un número =RAÍZ(número) =SQRT(number) 👉 Devuelve la raíz cuadrada de un número =ENTERO(número) =INT(number) 👉 Redondea un número hacia abajo hasta el entero más cercano =DECIMAL(número; decimales) =ROUND(number, num_digits) 👉 Redondea un número a la cantidad de decimales indicados =PI() =PI() 👉 Devuelve el valor de π =FACT(número) =FACT(number) 👉 Devuelve el factorial de un número

🧮 Funciones Estadísticas

=PROMEDIO(rango) =AVERAGE(range) 👉 Devuelve el promedio de un conjunto de valores =MEDIANA(rango) =MEDIAN(range) 👉 Devuelve el valor central de un conjunto de datos =MODA(rango) =MODE(range) 👉 Devuelve el valor que más se repite =VAR(rango) =VAR(range) 👉 Devuelve la varianza de una muestra =VARP(rango) =VARP(range) 👉 Devuelve la varianza de una población =DESVEST(rango) =STDEV(range) 👉 Desviación estándar de una muestra =DESVESTP(rango) =STDEVP(range) 👉 Desviación estándar de una población =FRECUENCIA(datos; clases) =FREQUENCY(data_array, bins_array) 👉 Distribución de frecuencias entre rangos =COEF.DE.CORREL(rango1; rango2) =CORREL(array1, array2) 👉 Devuelve el coeficiente de correlación

🔍 Funciones de Búsqueda y Referencia

=BUSCARV(valor_buscado; tabla; columna; [ordenado]) =VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup]) 👉 Busca un valor en la primera columna de una tabla =BUSCARH(valor_buscado; tabla; fila; [ordenado]) =HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup]) 👉 Busca un valor en la primera fila de una tabla =COINCIDIR(valor_buscado; rango; [tipo]) =MATCH(lookup_value, lookup_array, [match_type]) 👉 Devuelve la posición de un valor en un rango =INDICE(matriz; fila; [columna]) =INDEX(array, row_num, [column_num]) 👉 Devuelve el valor de una celda específica dentro de una matriz. (En el excel de respondus aparece "return" que es área, row y col son fila y columna, el parametro area es opcional

🧠 Funciones Lógicas

=SI(condición; valor_si_verdadero; valor_si_falso) =IF(logical_test, value_if_true, value_if_false) 👉 Ejecuta una acción según se cumpla o no una condición =SI.ERROR(valor; valor_si_error) =IFERROR(value, value_if_error) 👉 Devuelve un valor si hay error, otro si no lo hay =Y(cond1; cond2; ...) =AND(cond1, cond2, ...) 👉 Devuelve VERDADERO si todas las condiciones son verdaderas =O(cond1; cond2; ...) =OR(cond1, cond2, ...) 👉 Devuelve VERDADERO si al menos una condición es verdadera =ESBLANCO(valor) =ISBLANK(value) 👉 Verifica si una celda está vacía =ESERROR(valor) =ISERROR(value) 👉 Verifica si hay un error

🅰️ Funciones de Texto

=CONCATENAR(texto1; texto2; ...) =CONCATENATE(text1, text2, ...) 👉 Une cadenas de texto =ESPACIOS(texto) =TRIM(text) 👉 Elimina espacios innecesarios =IZQUIERDA(texto; num_caracteres) =LEFT(text, num_chars) 👉 Devuelve los primeros caracteres desde la izquierda =DERECHA(texto; num_caracteres) =RIGHT(text, num_chars) 👉 Devuelve los últimos caracteres desde la derecha =EXTRAE(texto; inicio; num_caracteres) =MID(text, start_num, num_chars) 👉 Devuelve un número específico de caracteres desde una posición =LARGO(texto) =LEN(text) 👉 Devuelve la longitud de un texto =MINUSC(texto) =LOWER(text) 👉 Convierte el texto a minúsculas =MAYUSC(texto) =UPPER(text) 👉 Convierte el texto a mayúsculas =NOMPROPIO(texto) =PROPER(text) 👉 Convierte el texto a formato título =SUSTITUIR(texto; viejo; nuevo; [ocurrencia]) =SUBSTITUTE(text, old_text, new_text, [instance_num]) 👉 Reemplaza un texto específico =REEMPLAZAR(texto; inicio; num; nuevo) =REPLACE(old_text, start_num, num_chars, new_text) 👉 Reemplaza parte del texto según posición =TEXTO(valor; formato) =TEXT(value, format_text) 👉 Da formato a un número o fecha =HALLAR(texto_buscado; texto; [inicio]) =FIND(find_text, within_text, [start_num]) 👉 Encuentra posición de texto (sensible a mayúsculas) =ENCONTRAR(texto_buscado; texto; [inicio]) =SEARCH(find_text, within_text, [start_num]) 👉 Encuentra posición de texto (no sensible a mayúsculas)

🧠 Funciones de Matrices

=MMULT(matriz1; matriz2) =MMULT(array1, array2) 👉 Multiplica dos matrices =MDETERM(matriz) =MDETERM(array) 👉 Devuelve el determinante de una matriz =MINVERSA(matriz) =MINVERSE(array) 👉 Devuelve la matriz inversa =TRANSPONER(matriz) =TRANSPOSE(array) 👉 Intercambia filas por columnas

📋 Funciones de Conteo y Sumatoria

=CONTAR(rango) =COUNT(range) 👉 Cuenta cuántas celdas tienen números =CONTAR.SI(rango; criterio) =COUNTIF(range, criteria) 👉 Cuenta celdas que cumplen una condición =CONTARA(rango) =COUNTA(range) 👉 Cuenta celdas no vacías =SUMA(rango) =SUM(range) 👉 Suma un conjunto de valores =SUMAR.SI(rango; criterio; [rango_suma]) =SUMIF(range, criteria, [sum_range]) 👉 Suma celdas que cumplen una condición =PROMEDIO.SI(rango; criterio; [rango_prom]) =AVERAGEIF(range, criteria, [average_range]) 👉 Promedia celdas que cumplen una condición

💡 Otros Operadores

^ 👉 Potenciación (Ej: 2^3 = 8) & 👉 Concatenación de textos (Ej: "Hola" & " Mundo" = "Hola Mundo")
